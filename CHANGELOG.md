# Changelog

All notable changes to the **Ambience** extension are documented here.
This project adheres to [Semantic Versioning](https://semver.org).

## [1.4.0]

### Added
- **Ambience: Amber** — a warm, low-contrast amber theme ported from the author's
  Amber desktop OS: soft amber on warm black, subtle borders, and an integrated
  terminal wired to the exact 16-colour Ghostty palette. Ships JetBrainsMono
  Nerd Font editor defaults.

### Removed
- Trimmed the collection to a focused 13-theme set: removed Hannah Montana,
  The Colour Out of Space, The Toxic Avenger, Spirited Away, Ledger,
  Ledger — Day, Solaris and Solaris — Day.

### Changed
- Reworked the README with a per-theme screenshot gallery.
- Added an extension icon and a gallery banner.
- Corrected the declared license to **GPL-3.0** (to match `LICENSE`/`NOTICE`).
- Refreshed the description and pruned keywords.

## [1.3.0]

### Added
- Paper, Ledger and Ledger — Day themes.
- `NOTICE` file documenting copyright scope and redistribution obligations.

### Fixed
- Transparent inserted/removed text borders in the diff editor.

## [1.2.0]

### Added
- The Empire Strikes Back theme; updated extension description.
